# SQL E-Commerce Analytics Project

## Overview
This project analyzes real-world e-commerce data (100k+ orders) using SQL.
The focus is on advanced SQL analytics, business insights, and data modeling.

## Dataset
- Brazilian E-Commerce Public Dataset by Olist
- Source: Kaggle
- Size: ~100,000 orders

## Tools
- PostgreSQL
- pgAdmin 4
- SQL (CTEs, Window Functions, Cohort Analysis)
- Power BI (visualization)

- What is customer retention by cohort?
