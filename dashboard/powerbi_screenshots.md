Power BI Dashboard Screenshots
==============================

This folder contains screenshots of the final Power BI dashboards built on top of the PostgreSQL analytics layer.

All metrics are calculated using SQL (mart schema) and Power BI DAX.
Screenshots represent the final analytical output of the project.
